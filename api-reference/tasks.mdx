## Overview

Tasks are units of work associated with fields. Use the Tasks API to automate task creation, assignment, and status updates.

## Endpoints

### List tasks

`GET /api/tasks`

Returns a paginated list of tasks, optionally filtered by field, status, or assignee.

### Get a task

`GET /api/tasks/{taskId}`

Fetch details for a single task by ID.

### Create a task

`POST /api/tasks`

Create a new task with a title, description, field, assignee, and due date.

### Update a task

`PATCH /api/tasks/{taskId}`

Update task properties such as status, due date, and assignee.
