## Overview

Fields are the core spatial unit in Fieldgrid. Use the Fields API to manage fields programmatically.

## Endpoints

### List fields

`GET /api/fields`

Returns a paginated list of fields the caller has access to.

### Get a field

`GET /api/fields/{fieldId}`

Fetch details for a single field by ID.

### Create a field

`POST /api/fields`

Create a new field with a name, location, and optional metadata.

### Update a field

`PATCH /api/fields/{fieldId}`

Update mutable properties of an existing field.

## Authentication

All endpoints require authentication (for example, bearer tokens or API keys). See the main API introduction page for details.
