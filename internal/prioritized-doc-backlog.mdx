## How to use this backlog

- When planning work:
  - Choose items from the earliest phase that still have gaps.
- When building new features:
  - Add or update docs in the appropriate section and phase.
- When discovering repeated questions:
  - Consider promoting those topics into Phase 1 or 2 as dedicated guides.

This backlog should evolve as Fieldgrid evolves, but it gives us a shared starting point for where to invest documentation effort next.
